Welcome to Eek's Attic

Eek's Attic is different than what you're used to. I've been a grown woman for a long time now, and almost every single women's retail store I've ever been to focuses on one thing: profit. I get that. It's a business. But I also see this huge group of my fellow women who hate themselves for one reason or another and who treat each other horribly. I want to change that.

eeksattic.com is a real website my wife and I have been operating for almost three years now. This is our mission statement above. I've been exposed repeatedly to the source code of this website. This final project was an opportunity for me to breathe more life into a project I have been working on for some time now. I'm making an effort to gamify the landing site for eeksattic.com. This is a rough foundation for that project that has been converted into a retail website for this project.
